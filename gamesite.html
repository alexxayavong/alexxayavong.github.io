Game

This is the game I made based on the anime Demon Slayer. This game was made using Processing (the same app used for the self-portrait). The game is basically move the mouse to hit the target. I decided to use Demon Slayer because I was thinking that you can basically go through the whole story of the show. Each level is an arc in the show so each enemy is based on the level and what arc they appeared in.
